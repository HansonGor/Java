### 一. [DOS常用命令](https://blog.csdn.net/weixin_43306844/article/details/86615870)
```

mkdir abc // 创建abc目录


d: e: // 进入d盘,e盘

dir  // 查看当前目录有什么文件和文件夹

cls // 清屏

cd 



```

### 二. ipconfig：查看ip配置

```

 ipconfig
 ipconfig/all // 更多IP配置

```

### 三. ping:[ ip地址/域名] - 看是否能访问服务器

```

ping www.baidu.com

ping www.baidu.com -t // 检查网路是否稳定

```
